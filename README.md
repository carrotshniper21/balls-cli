## Installation
### Prerequisites:

- [`mpv`](https://mpv.io) - Player used to play the video
- [`cargo`](https://github.com/rust-lang/cargo) - For compiling the program


### Linux Install
```
- cargo build --release && cd target/release && sudo cp balls-cli /usr/local/bin
```

# Usage
```
balls-cli [movie name]
```
